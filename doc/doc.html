Development Steps

  1. To do the login
  2. After authenticated, depending of the user profile will be the options that must be appear

Government profile

  1. Create a contract:
    • First of all we have installed truffle $sudo npm install -g truffle and we have followed the instructions of [1].
    • First of all we must create a folder and within it, we must initialize using the following: $ npm init -y
  2. List contracts:
    • Generate a list of contracts created by the government.
  3. Add manufacturers:
    • This function contains a form receiving a name and an email (source addManufacturers).
    • Values name and email must save within the blockchain.
